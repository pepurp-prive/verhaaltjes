# **App Name**: World Weaver

## Core Features:

- Story Generation: Generates a unique story based on a user-provided prompt, considering selected character details as a tool for incorporating specific information in the output.
- Character Creation: Allows users to create detailed character profiles with various attributes such as name, age, appearance, clothing, and personality, leveraging AI for generating suggestions.
- Character Management: Enables users to save, edit, and manage their created characters for use in multiple stories.
- Story Overview: Presents an overview of saved stories, allowing users to read, edit, and continue their narratives.
- AI Category Fill: Assists the user in automatically filling fields in character creation using AI for various sections like basic data, appearance, clothing and details.
- AI Field Generation: Supports individual field content generation such as character names, ages, job, story prompts, striking features using the built-in AI assistant.

## Style Guidelines:

- Primary color: Deep forest green (#228B22) to reflect creativity and nature, grounding the digital space.
- Background color: Soft sage (#D3DBC0) as a muted backdrop to enhance readability and a sense of calm.
- Accent color: Warm terracotta (#E07A5F) to draw attention to important actions.
- Body font: 'Alegreya', serif, for an elegant, intellectual feel
- Headline font: 'Belleza', sans-serif, aligned to art and design.
- Use simple, outlined icons with rounded corners to maintain a friendly, approachable aesthetic.
- Subtle transitions and loading animations to provide feedback and enhance user experience without overwhelming the interface.
